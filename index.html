NFePHP

Nota fiscal eletrônica em PHP - gratuita e aberta

Download this project as a .zip file Download this project as a tar.gz file

OBS: Para saber a diferença entre o NFePHP do http://www.assembla.com/spaces/nfephp/ e do GitHub, veja nossa FAQ e tire suas dúvidas.

O NFePHP é um projeto de software livre para Nota Fiscal Eletrônica. Na versão atual ele já realiza as principais tarefas relacionadas a NFe, tais como:

  • Geração da nota fiscal em XML a partir de TXT;
  • Assinatura da nota utilizando certificado digital e-CNPJ e e-NFe A1 (em PHP nativo!);
  • Envio e validação das notas em lotes por web services;
  • Geração da DANFE;
